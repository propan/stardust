# STARDUST [CLIENT]

a client for multiplayer version of Asteroids written in ClojureScript

[Live Demo] [1] 

## Usage

** REQUIRES A RUNNING STARDUST SERVER**

To build the project, execute

```bash
lein cljsbuild auto dev
```

and open `index.html` in your browser.
